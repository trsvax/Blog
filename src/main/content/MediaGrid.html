Overview

Today is going to be easy. I'm just going to create a media grid component by wrapping the Tapestry loop component.

Requirements

  • Follow semantics of Tapestry grid but for media

Implementation

This is just a wrapper for the Tapestry loop component with the right css classes for displaying the content in a Bootstrap template. For this I'll need a component class and a template. As you can see not much going on here. The component just passes it's parameter and body on to the loop component.

Conclusion

So why did I bother with this since it's so simple. First I don't have to remember the semantics for a Bootstrap media grid. Secondly if I decide to use some other template I can just substitute a different media grid component. And most important this component is about to get much more complicated by integrating the pagination component to support paging.
